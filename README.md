# SimpleKeyboardMouseHook

Use WASD to move the cursor.  
Use the Space Bar to simulate mouse down and mouse up.  
Use Ctrl + Space Bar to simulate a right click.  
Use Shift + WASD to scroll vertically or horizontally.  

# Use
After installation or opening the app.  
There will be an icon in the System Tray.
Right-click the icon to show more options.

# Hotkey
Alt + F3 : Toggle the hook on and off.
